# 편의점 음식 레시피 정보 플랫폼


## Rescue9 GIT commit 규칙 0.1
- 커밋메세지: <커밋 행동에 대해 명시>-<커밋내용>
  - 시작: 새로운 기능(앱)의 추가
  - 수정: 기존 존재하는 파일의 수정
  - 추가: 새로운 파일을 추가하는 경우
  - 삭제: 기존의 파일을 삭제하는 경우
  - ex) 추가-recipesDetailPage
- branch 생성
  - feature/<본인이 작업한 내용 중 가장 핵심이 되는 내용>을 적는다
  - ex) feature/recipesDetailPage
- pull request에 본인이 작업한 내용을 간략히 작성한다
